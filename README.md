# iCalcUIKit
# SwiftCalc UIKit

A clean and fully functional Calculator app built using UIKit in Swift.

## 📱 Features
- Basic arithmetic operations (+, −, ×, ÷)
- Clean and responsive UI
- Auto Layout with programmatic constraints
- Smooth user interaction
- Error handling (division by zero, etc.)

## 🛠️ Built With
- Swift
- UIKit
- Auto Layout (Programmatic UI)

## 🎯 Purpose
This project was built to strengthen my understanding of UIKit, layout handling, and building real-world iOS applications from scratch without Storyboards.

## 🚀 Future Improvements
- Scientific calculator mode
- Dark mode support
- History tracking
- Animations and UI enhancements
